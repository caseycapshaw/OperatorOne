Usage Rules

Do
Use on dark backgrounds (#0a0a0f to #12121c)
Maintain clear space equal to badge height / 4
Use hover glow for interactive contexts
Scale border weight and font weight together
Use the light-bg variant on white/light surfaces
Don't
Apply static glow — glow is hover/active only
Place on mid-tone backgrounds (gray zones)
Stretch or distort the badge proportions
Change the O/1 color relationship
Add drop shadows outside the glow system

Components

Core UI patterns. All components use borders over shadows, neon highlights over fills, and the glow system for interactive states.

06

Voice

Calm, competent, direct. The person in the chair who's already handled it. Not shouting, not selling — just stating facts.

Principles

Terse
Say it in fewer words. "Done." not "Your action has been completed successfully!"
Confident
"Systems online." not "We believe everything should be working now."
Capable
Treat the user as competent. Never condescend.
No Exclamation
Never "Oops!", "Yay!", or "Great job!" Period.
