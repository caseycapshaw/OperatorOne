# Code of Conduct

## Our Standards

We are committed to providing a welcoming and respectful environment for everyone. We expect all participants to:

- Be respectful and constructive in communication
- Accept constructive feedback gracefully
- Focus on what is best for the project and community
- Show empathy toward other contributors

## Unacceptable Behavior

- Harassment, trolling, or personal attacks
- Publishing others' private information without consent
- Any conduct that would be considered inappropriate in a professional setting

## Enforcement

Project maintainers may remove, edit, or reject contributions that violate this code of conduct. Repeated violations may result in a ban from the project.

## Scope

This code of conduct applies to all project spaces, including issues, pull requests, and discussions.
